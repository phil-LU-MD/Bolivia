Se filtran las circunscripciones 38 y 39 (norte del departamento de Potosí) y se calcula el porcentaje de voto nulo por mesa. Solo se consideran las mesas donde el voto nulo representa 50% o más de los votos emitidos. En total, 242 mesas (el 37% de las 639 mesas en estas dos circunscripciones) cumplen con este umbral.

resultados_potosi <- resultados_bol25 %>%
  filter(CodigoCircunscripcionU %in% c(38, 39)) %>% 
  filter(Descripcion == "PRESIDENTE") %>% 
  mutate(PorcentajeNulo = round((TotalVotoNulo / VotoEmitidoReal) * 100,0)) %>% 
  filter(PorcentajeNulo >= 50) %>% 
  arrange(desc(PorcentajeNulo)) %>% 
  mutate(Clasificacion = row_number()) %>% 
  select(Clasificacion, everything())  # Move Clasificacion to the very left
